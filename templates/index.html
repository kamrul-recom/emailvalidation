POST/api/validate
Synchronous — validates up to 5,000 emails inline and returns results immediately. No polling needed.
Request
curl -X POST http://localhost:5000/api/validate \ -H "Content-Type: application/json" \ -d '{ "emails": ["alice@gmail.com", "fake@mailinator.com", "bob@company.io"], "check_dns": false }'
Response
{ "total": 3, "stats": { "valid": 2, "invalid": 1, "valid_pct": 66.7, ... }, "results": [ { "email": "alice@gmail.com", "legitimacy": "valid", "activity": "active", "score": 69, ... } ] }
POST/api/upload
Async — for large batches. Upload any file format, get a job_id, then poll for results.
Request
curl -X POST http://localhost:5000/api/upload \ -F "files=@emails.csv" \ -F "check_dns=true" # → { "job_id": "abc-123", "total": 50000 } # Poll: curl http://localhost:5000/api/job/abc-123
POST/api/paste
Submit raw text as JSON — same as the paste tab but programmatic.
curl -X POST http://localhost:5000/api/paste \ -H "Content-Type: application/json" \ -d '{"text": "alice@test.com\nbob@corp.io", "check_dns": false}'
